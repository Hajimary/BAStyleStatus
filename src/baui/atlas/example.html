使用说明

  1. 运行 npm run generate:atlas 生成CSS和TypeScript文件
  2. 在HTML中引入生成的CSS文件：<link rel="stylesheet" href="dist/baui/CommonUI.css">
  3. 使用两个类名来显示精灵：
    • 基础类：commonui-sprite 或 emoji-sprite
    • 具体精灵类：commonui-common-0 或 emoji-emoji-sad
  4. 示例：<div class="commonui-sprite commonui-common-0"></div>
